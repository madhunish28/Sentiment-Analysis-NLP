# Sentiment Analysis using NLP

## 📌 Project Overview
This project analyzes text data and classifies it into positive or negative sentiment using machine learning.

## Problem Statement
Predict sentiment from text reviews using Natural Language Processing (NLP).

## Tools Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer

## Model Used
- Naive Bayes (MultinomialNB)

## Steps Performed
1. Data Loading
2. Text Preprocessing
3. TF-IDF Vectorization
4. Train-Test Split
5. Model Training
6. Model Evaluation

## Results
- Achieved good accuracy in sentiment classification

## Model Insights

1. Used Naive Bayes model, which is highly effective for text classification tasks.

2. The model achieved good accuracy, indicating it can correctly classify sentiments.

3. TF-IDF vectorization helped improve model understanding of important words.
